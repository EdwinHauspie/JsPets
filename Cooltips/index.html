Cooltips – jQuery extension

Requires

  • jQuery
  • jQuery UI

Usage

<div data-cooltip="Hello world">Hello</div>

Demos

Default : ) data-cooltip="Hello world"
Html : ) data-cooltip="&lt;b&gt;Bold text&lt;/b&gt;"
Element : ) data-cooltip="#dog"
Function : ) data-cooltip="function() { return navigator.userAgent; }"
Sticky : ) data-sticky-cooltip="#form"
